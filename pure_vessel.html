SAVES DE CLASSE & DESAFIOS

💻 Como instalar no PC (Windows)

1. Pressione
Windows + R, digite o
comando e dê Enter:

%AppData%\..\LocalLow
\Team Cherry\Hollow Knight

2. Backup: Copie seus arquivos user.dat para
outra pasta.
3. Instalação: Cole o arquivo baixado na
pasta.
4. Nota: O número no nome (user1, user2...)
define o slot no jogo.

📱 Como instalar no Mobile
(Android)

Android > data > com.TeamCherry
.HollowKnight > files

1. Substituição: Mova o .dat baixado para
dentro da pasta files.
2. Nome: O arquivo deve se chamar
user1.dat
para funcionar.
3. Android 11+: Clique em "Usar esta pasta"
quando o aviso aparecer.

⚠️ Atenção aos nomes: O arquivo deve estar nomeado exatamente como user1.dat para o jogo reconhecer.
